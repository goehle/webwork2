Possible Errors

  • AttributeError - This occurs when there is a problem calling methods or using using/assigning fields within an object.
  • IndexError - This occurs when a subscript (a.k.a. index) is out of range.
  • NameError - This occurs when you try to reference a variable that has not been defined.
  • SyntaxError - This occurs when you have incorrect syntax stopping the python program from running.
  • TypeError - This occurs when your try to use a variable of an incorrect type for a specific operation. Also apply when a function/method expects an argument(s) and none are supplied.
